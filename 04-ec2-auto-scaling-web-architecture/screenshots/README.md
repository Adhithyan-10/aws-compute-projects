# Project Implementation Screenshots

This folder contains screenshots captured during the implementation of the **Auto Scaling Web Architecture using EC2 and Application Load Balancer**.

The screenshots illustrate the step-by-step configuration of AWS services used in the project.

## Screenshot Overview

The images included in this folder demonstrate the following stages of the project:

1. Launch Template creation for EC2 instances
2. User Data script used to configure Apache web server
3. Auto Scaling Group configuration
4. Auto Scaling activity and instance replacement
5. EC2 instances managed by the Auto Scaling Group
6. Application Load Balancer configuration
7. Target Group health check status
8. Load Balancer response from EC2 instance
9. Load Balancer response from another EC2 instance

These screenshots provide visual confirmation of the architecture working successfully.
