# Project Demo Videos

This folder contains demonstration videos for the **Auto Scaling EC2 Architecture using Application Load Balancer** project.

The videos show the implementation and working of the scalable infrastructure built on AWS.

- Auto Scaling Group setup

- EC2 instance deployment
